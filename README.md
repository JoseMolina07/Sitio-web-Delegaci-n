# 🌐 Sitio Web Delegación Tapijulapa

## 📌 Descripción
- Proyecto desarrollado durante mi formación en educación media superior (CECyTE)
Desarrollo y mantenimiento de un sitio web informativo para la delegación de Tapijulapa, Tabasco.
El objetivo del proyecto fue proporcionar una página sencilla para mostrar información básica a la comunidad.

## 🧠 Mi participación

* Desarrollo inicial del sitio web
* Mantenimiento durante aproximadamente 1 año
* Corrección de errores y ajustes menores
* Actualización de contenido

## ⚙️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript
* Python (uso básico en algunas funcionalidades)

## 🧩 Funcionalidades

* Visualización de información general
* Navegación simple entre secciones
* Diseño responsivo básico

## 🛠️ Retos y aprendizajes

* Comprender cómo mantener un sistema activo durante un periodo prolongado
* Resolver errores en un entorno real
* Mejorar la organización del código
* Aplicar validaciones básicas en formularios

## 📈 Alcance del proyecto

Este fue un proyecto sencillo, enfocado en funcionalidades básicas y mantenimiento continuo.
No se implementaron características avanzadas, pero permitió adquirir experiencia práctica en desarrollo web y soporte en producción.
